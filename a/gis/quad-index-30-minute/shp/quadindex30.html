    Title: 1:100000 scale usgs quad map index

Description  
    Abstract
    This theme shows the extent and name of each of the 1:100000 scale
    quadmaps for the state of Oregon.
     
     
    Purpose
    This theme is used to delineate the boundarys of the 1:100k scale quad maps.
     
     
Supplemental_Information
    Procedures_Used
    The origin of this theme prior to August 28,1991 is unknown.  It appears
    that the data was created by the USGS.  The theme was imported
    and projected from state plane north to a lambert projection.  The theme
    was than renamed and built for polygon topology.  The externalall
    command was run on the theme in 1991 and again in 1993.  (The externalall
    command finds all info subdirectories and corrects the Arc linkages to
    the tables.  In 1994 the theme was copied to its current name and exported
    out to the SSCGIS ftp site.  In early 1997 the theme was imported in and
    projected into the Oregon transfer standard.  The theme was built for
    polygon topology and put back out on the SSCGIS ftp site.  The theme was
    imported in late 1997 so the labelerrors command could be run on it.
    Metadata was created for the theme and it was put back out on the SSCGIS
    ftp site.
     
     
    Revisions
    No revisions
     
     
    Reviews_Applied_to_Data
    The theme was checked for labelerrors.  Earlier reviews were not recorded.
