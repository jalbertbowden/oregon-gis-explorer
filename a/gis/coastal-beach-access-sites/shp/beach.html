Citation_Information 
    Originator: Oregon Department of Parks and Recreation
    Publication_Date: March 31, 1989
    Title: Inventory of Oregon Coastal Beach Access Sites
    Geospatial_Data_Presentation_Format: Database
Description  
    Abstract
    The beach access points theme was created to display the location
    of public beach access along the Oregon Coast.  Boat ramp
    locations were added to the dataset to allow users to view the
    location of boat ramps along the Columbia River and the
    Willamete River north of the Oregon City Dam.
    Purpose
    The theme allows the user to locate and extract information on
    various beach access locations.  The information includes
    approximate geographic location and attributes defining the
    development level of the site.
Supplemental_Information
    Procedures_Used
    The data was gathered from two different sources.  The items in the
    data base from area to Parking-Added came from a report created for
    OPRD by The Benkendorf Associates Corporaton.  The site locations
    were derived from various state, local and private sources.
    Information sheets were derived for each identified location.  The
    data was placed into a lotus spread sheet.  The data was than sent
    out to the local sources who checked it for accuracy.  The second
    group of attributes (items after Parking-Added) were supplied by the
    Oregon Marine Board with assistance from the Oregon Department of
    Fish and Wildlife.  The information gathering and presentation was
    funded in part by the US fish and wildlife Sport Fish Restoration
    Program.  In January of 1991 the State Service Center for GIS
    (SSCGIS) made a blank coverage by using the Arc/Info Create command
    getting the projection and tic locations from another theme.  The
    theme was edited using Arc/Infos ArcEdit modual.  (As with much
    of the history of this coverage no documentation was kept for what
    was edited.  Due to the creation of point topology it is assumed
    most of the editing involved the addition of points.)  The theme
    was built for polygon topology.  The theme was edited several more
    times in arcedit.  Once the externalall command was run in order to
    correct the file path names.  Arcedit was run again and the theme
    was again externalled.  The theme was built for point topology and
    exported out in Februrary of 1991.  In March of 1991 the theme was
    copied to a new workspace and editted several time over the next
    month.  The externalall command was run and the theme was again
    edited.  Point topology was built for the theme, it was externalall
    to correct directory names and it was again built for topology.  The
    theme was edited built for topology and copied to new workspaces
    several times over the next month.  The theme was renamed back to its
    original name in May of 1991 (it had been copied to another name a
    few weeks previous to this time.  The theme was built for point
    topology and again edited.  The theme was edited built for topology
    and externalalled several times during the next few months.
    (NOTE: The SSCGIS assumes that the 1990 log file entry dates are a
    mistake.  They should say 1991.)  The theme was projected to UTM
    coordinates and built for point topology.  The theme was copied to
    a new directory and externalalled to correct the path names.  The
    theme was than copied to a new name.  The theme was exported out
    and the path names were corrected using the externalall command.
    The theme was than renamed twice and the externalall command was
    run on it.  The theme was built for point topology and left alone
    for six months.  It was externalled and built for topology again.
    The theme was renamed to its current name and built for topology
    in June of 1992.  The subdirectory pathnames were corrected
    using the externalall command.  The theme was projectected in
    March 1993 to a lambert projection.  The theme was exported
    for use on the SSCGIS ftp site in April of 1995.  The theme
    was imported in February 1997 so it could be projected into the
    current state data transfer standard (lamber international feet.)
    The theme imported in during December 1997 so metadata could be
    created for it.  The metadata was finished in February 1998 (after
    documents were obtained from OPRD (Oregon Parks and Recreation).
    The theme was exported and put on the sscgis ftp site.

    Items from area to parking-capacity were provided by a consultant
    report done for OPRD and DLCD.  The items after parking-capacity
    come from the Oregon Marine Board.  Points along the coast and
    along the Columbia River in Clatsop county came from the OPRD-
    DLCD report.  The points along the Columbia River and the
    Wilamette River come from the Oregon Marine Board.  Wilamette
    River is limited to points north of the Oregon City Falls.
