    Title: Oregon Graticule

Description  
    Abstract
    This theme is a one degree latitude and longitude graticule for the
    state of Oregon.
     
     
    Purpose
    This theme was generated in order to create a 1 degree by 1 degree
    gradicule for the state of Oregon.
     
     
Supplemental_Information
    Procedures_Used
    This dataset is a legacy dataset.  Current SSCGIS staff were not present
    when it was created.  No descriptions were left of what was done in
    each of the ARCEDIT session.  The theme was created using the generate
    command in Arc/Info.  Thirty seperate arcedit sessions were undertaken
    to create this in July of 19922 (believed to primarly be the generate
    command.)  The theme was built for line topology and the theme was
    edited 10 more times.  The coverage was exported.  In April of 1993
    the extenalall command was run.  The theme was than copied into the
    SSCGIS data library.  Once in the library it was exported for use on
    the SSCGIS ftp site.  In February of 1997 the theme was exported in
    and projected to the Oregon transfer standard projection.  The theme
    was than built for line topology and exported back out.  It was imported
    in during December of 1997 for the creation of metadata.
     
     
    Revisions
    Orginal Version
     
     
    Reviews_Applied_to_Data
    The theme was lined up over other Oregon data layers to make sure the
    graticule approximated actual latitude and longitude lines.
