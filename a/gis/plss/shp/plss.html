    Title: Public Land Survey System

Description  
    Abstract
    This theme contains PLS lines for the State of Oregon.  This
    Pls theme includes donation claims lands.  Attributes in this
    theme show Township Range and Section values.
     
     
    Purpose
    The purpose of this theme is to delineate the Oregon Public Land
    survey lines including the donation land claim parcels.
     
     
Supplemental_Information
    Procedures_Used
    The PLSS theme was created from USGS 1:100,000 DLGs.  No records
    exist of the process used to convert the DLG files into Arc/Info
    format.  In May of 1994 the service center dissolved the existing
    coverage on the loc-dlc item.  Once the dissolve was complete the
    theme was cleaned using a dangle length of 1.5 and a fuzzy-tolerance
    of 1.5 using the poly option.  The next step was using the centroid-
    labels command with the inside option.  (This moved the polygon
    labels to the center of the polygon.) The build command was used to
    create topology after the labels were moved.  The addxy command was
    used to create the x-coord and the y-coord items.  The cover was
    exported three times.  The external command was run on the theme to
    correct the info file.  The theme was than exported a fourth time.
    The PLS theme was copied to another work directory where it was
    exported 4 more times over the ensuing 3 years.  In February of
    1997 the them was imported and projected into the state transfer
    standard.  Once the theme was project it had topology built.
    Final the theme was imported in again so metadata could be created.
     
     
    Revisions
    The current version has not been revised
     
     
    Reviews_Applied_to_Data
    This theme is a legacy data theme.  No current members of the SSCGIS or
    the OWRD theme were around when it was created.  The data table has
    been spot checked, but the other checks that occured are unknown.
